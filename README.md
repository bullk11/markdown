# Markdown Project

Markdown with simple functionalities

## Getting Started

Need internet connection for the libraries.

```
Download the project
```

```
Open markdown.html
```

### Usage

```
*text* to make it italic
```
```
**text** to make it bold
```
```
#text# to make it h1
```
```
###text## to make it h2
```
```
###text### to make it h3
```

To save the Markdown press button "Save".
